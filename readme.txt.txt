Argumentation-theory-toolkit
Argumentation theory visualisation toolkit on JS with D3js

•Copy all repository files to your server
•Open index.html in your browser

graph.json - Data, if you need to change data you can change just this file
Or you can upload new file with Upload button, after that file graph.json will be rewritten
Then you need to refresh browser page.
